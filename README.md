ansible-role-eclipse
=========

An ansible role that installs an configures an Eclipse IDE with Java EE support
and some tweaks to improve performance.

Requirements
------------

- Java JDK


Example Playbook
----------------

You can use it like:

```yml
- hosts: servers
  roles:
    - caarlos0.eclipse
```
